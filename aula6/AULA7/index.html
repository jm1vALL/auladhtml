aula 07 - estudo de css

A sigla css e cascading style sheets é uma linguagem utilizada para estilizar paginas da web. utilizando esta linguagem podemos definir:

  • cor de fund, texto e elementos.
  • fonte e tamanho das letras
  • posicionamento de elementos dentro da pagina
  • efeitos visuais e animações

O css funciona em conjunto com html, enquanto o html diz o que aparece, o css diz como deve aparecer.
